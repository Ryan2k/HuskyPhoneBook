Husky Phone Book

Instructions

1. To get all collected information click "Load Data"

2. When this is pressed, the data will load to your screen and 2 buttons will appear

3. The first is "clear data" which returns you to the homepage

4. The second is a query form which needs to take in either a first name, last name, or both

5. By pressing submit, you will recieve only data relevant to your query parameters

6. If there is information collected for a criteria for a person, it will be highlighted

7. Otherwise, it means that there is no information on the person regarding the columns criteria

Query

Search for a person by inputting their first name, last name, or both, then press "Search"
